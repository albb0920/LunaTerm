快速上手
  • Ctrl 組合鍵:
    按一下壓軌跡球 (可能是 OK 鍵或中心鍵)或者 搜尋鍵，再按下鍵盤按鍵
  • 可自訂功能按鈕:
    連線時按下 音量升高鍵 叫出，可在設定中增加
  • 軟體鍵盤: 連線時按下 音量降低鍵 可以顯示或關閉
  • 快速切換連線: 連線時按 MENU 選擇
  • 放大鏡: 長壓畫面
  • 操作手勢: U-上 D-下 L-左 R-右
    操作手勢
    方向鍵 上U
    方向鍵 下D
    方向鍵 左L
    方向鍵 右R

    RLR
常見問題
Q: 我要怎們輸入帳號密碼
A: 連線時以音量減低鍵叫出虛擬鍵盤輸入，或者於站台列表長壓編輯設定
Q: 使用 Wi-Fi 時，閒置會自動中斷
A: 請檢查 Wi-Fi 閒置設定
關於

LunaTerm 是一個應台灣 BBS 使用者使用習慣誕生的開源專案，原始碼以 rTerm 為基礎。

本程式原始碼使用 GPLv3 授權，可以在 這裡 找到授權條款，本專案原始碼可在 專案網站 中找到。

本專案一共使用的原始碼如下:

  • rTerm: 本專案的基礎
    授權方式: GPLv3
  • ConnectBot: 另一個 Android 上的 Terminal，rTerm 參考之一
    授權方式: Apache License 2.0
  • JTA Telnet/SSH client: JAVA 的 SSH/Telnet 函式庫，rTerm 與 ConnectBot 的基礎
    授權方式: GPLv2
